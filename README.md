# API測試項目

## 下載Repo
`$ git clone https://github.com/Kouei-Lin/my-python-api`

## 進文件夾
`$ cd my-python-api`


`$ cd add_api`

## 安裝套件

`$ pip install -r requirements.txt`

[沒Python於此下載](https://www.python.org/downloads/)

## Server 跑項目
`$ python3 app.py`

## Server .env
如果有`.env_example`則需要編輯各項目所需`.env`

`$ cp .env_example .env`

`$ vim .env`

## Client 跑指令
複製貼上`command.txt`中的範例指令，`IP`改成伺服器，執行。
