# API測試項目

## 下載Repo
`$ git clone https://github.com/Kouei-Lin/my-python-api`

## 進文件夾
`$ cd my-python-api`


`$ cd add_api`

## 安裝套件

`$ pip install -r requirements.txt`

[沒Python於此下載](https://www.python.org/downloads/)

## .env
編輯各項目所需`.env`

`$ cp .env_example .env`

`$ vim .env`

## 跑項目
`$ python3 app.py`
